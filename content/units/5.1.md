# Финальная миссия 5.1 — Одностраничный сайт бренда

Собери одностраничник для вымышленного или реального бренда в стиле любимых упаковок.

**Чеклист**

1. Структура секций и сетка — как на коробке: hero, преимущества, линейка, контакты.
2. Токены цвета/типа из модуля 2 — без «случайного» HEX в разметке.
3. Либо статические данные, либо таблица в базе — но без ручного дублирования карточек.
4. Деплой или доступ по LAN — чтобы можно было показать результат без Фигмы.

**Выпуск:** обнови манифест в профиле одной фразой: что теперь для тебя «код».
