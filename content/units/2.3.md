# Миссия 2.3 — Просим ИИ правильно

Главное правило: один запрос = одна правка.

**Что сделать**

1. Сделай запрос только про отступы.
2. Сделай второй запрос только про кнопку.
3. Сделай третий запрос только про размер текста.

**Проверка себя**

- После каждого запроса меняется только одна часть.

**Зачем это нужно**

Так ты быстро понимаешь, какой запрос дал хороший результат.
