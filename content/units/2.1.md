# Миссия 2.1 — Первый промпт

Сделай скрин своей карточки. Открой Cursor — это редактор, где ИИ пишет код рядом с тобой.

**Задача**

1. Вставь шаблон промпта из подсказки тьютора (нижняя строка).
2. Добавь ограничения: Tailwind, без тяжёлых библиотек, один файл или компонент — как тебе удобнее.
3. Сохрани результат в проект; если что-то «поехало», второй промпт только про правку отступов.

**Почему это важно:** ты уже не перерисовываешь макет руками — ты задаёшь контракт, ИИ делает черновую сборку.
