# Миссия 3.2 — Переключение вариантов

У товара есть 3 вида. Человек должен переключать их без перезагрузки.

**Что сделать**

1. Добавь 3 кнопки варианта.
2. По нажатию меняй картинку товара.
3. Подсвечивай активный вариант.

**Проверка себя**

- Видно, какой вариант выбран сейчас.

**Зачем это нужно**

Так пользователь быстрее выбирает товар.
