# Миссия 2.3 — Промпт-инжиниринг для дизайнера

Один запрос — одна задача: анимация появления, фикс отступа «как в Figma», стиль кнопки с референс-сайта.

**Задача**

1. Составь три коротких промпта под три разные правки (не смешивай в один простыню).
2. Для каждого добавь входные данные: скрин, число отступа в px, ссылку на референс.
3. Отметь, какой ответ пришлось уточнить вторым сообщением — это нормально.

**Почему это важно:** диалог с ИИ как ревью с арт-директором: цикл «показать → поправить».
