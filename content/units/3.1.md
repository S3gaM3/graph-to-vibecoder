# Миссия 3.1 — Кнопка «Добавить»

Нужно простое поведение: нажал кнопку — увидел сообщение «Добавлено».

**Что сделать**

1. Добавь кнопку.
2. После нажатия покажи короткое сообщение.
3. Через пару секунд сообщение можно скрыть.

**Проверка себя**

- Нажатие работает каждый раз.

**Зачем это нужно**

Это первый живой шаг, когда экран реагирует на человека.
